**ANOVA analysis of neuromasts under 3 different treatments: WT, 40LY, and 400LY**
- Heatmaps of a neuromast to visualize gene expression can be made through heatmaps.py, this needs gene expression files and their matching segmentation
    - This also creates an intensity CSV
- This intensity CSV can be used in calculate_mean_intensity.py, which calculates the mean intensity for one neuromast
- Based on the neuromast intensity, an ANOVA can be run to test difference between treatment groups
